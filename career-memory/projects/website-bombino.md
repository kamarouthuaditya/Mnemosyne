Public marketing + utilities website for **Bombino Express**, an international courier / freight /
customs-clearance company. Built on a customized Next.js (App Router) stack.

Visitors submit several kinds of forms — contact, shipping quote, pickup request, customs-clearance
request, and KYC (identity/compliance upload). Each submission triggers transactional email and is
logged to Google Sheets. The **Mail Automation** feature owns that email pipeline.
